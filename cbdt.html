ROLE UX/UI Designer
UX Design, Design Thinking, User Flows, Wireframing, Prototyping, UI Design, Figma, Photoshop, Illustrator

Feb 2024
Problem Statement Banking institutions face challenges in managing bulk transactions efficiently, with manual processes leading to delays, errors, and operational complexities.
Solution The CBDT Bulk Transactions website streamlines banking processes by offering a secure platform for Host to Host (H2H) transactions. It features hierarchical access levels for different banking entities, file upload functionalities for tax submissions, real-time status updates for transaction approvals, and comprehensive reports and analytics for agency banks, non-agency banks, and corporates.
My Contribution As a UI/UX Designer, I led the design efforts to create an intuitive and user-friendly platform for CBDT Bulk Transactions. I conducted thorough user interviews and usability testing to gather insights, created detailed wireframes and user flows to map out the user journey, designed interactive UI elements for seamless data entry and analysis, and collaborated with developers to ensure a cohesive and functional web interface. My designs prioritized security, scalability, and user experience for banking professionals using the CBDT Bulk Transactions tool.
User Flow The user flow for CBDT Bulk Transactions illustrates the journey of users, including agency banks, non-agency banks, and corporates, through accessing the platform, uploading tax files, receiving approvals, and generating reports.
Wireframes Wireframes for CBDT Bulk Transactions detail the layout and functionality of different screens, including access-based login screens for different user roles, file upload interfaces, status tracking, and analytics dashboards.
User Mapping User mapping diagrams depict the hierarchical access levels and permissions for different user categories within the CBDT Bulk Transactions platform.
High-End Prototype The high-end prototype showcases the fully interactive and functional aspects of the CBDT Bulk Transactions platform, including real-time data updates, report generation, and user-specific dashboards.
